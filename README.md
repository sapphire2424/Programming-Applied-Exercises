# Programming Applied Exercises (C++)

「プログラミング応用演習」の講義にて取り組んだ演習課題およびレポート課題のソースコードを週ごとにまとめています。

## 概要
C言語の基礎を踏まえ、C++を用いたより高度なデータ構造の扱いやアルゴリズムの実装、メモリ管理（動的確保）などのコンピュータサイエンスの基礎を13週(第6週,第13週は復習)にわたって学習・実践しました。

## 学習・実装した主な技術項目
この演習を通じて、以下のトピックについて理論の理解とコードへの落とし込みを行いました。

### 1. C++の基礎と応用
- ポインタの深い理解と動的メモリ確保（`new`, `delete`）
- 構造体およびクラスを用いたデータカプセル化
- ファイル入出力（テキスト・バイナリデータの扱い）

### 2. アルゴリズムとデータ構造
- **探索アルゴリズム**: 線形探索、二分探索
- **ソートアルゴリズム**: バブルソート、選択ソート、クイックソート
- **線形データ構造**: リスト、スタック、キューの実装と操作
- **再帰処理**: 階乗計算やハノイの塔を用いた再帰ロジックの理解

### 3. 数値計算・応用プログラム
- 複素数計算クラスの実装
- 統計処理プログラム
- 簡易的なゲームロジックの構築

## リポジトリの構成

- `Week XX/`

## 開発・実行環境
- **言語**: C++
- **コンパイラ**: g++
- **OS**: Windows

## ひとこと
「プログラミング応用演習」を通じて、単に動くコードを書くだけでなく、計算効率やメモリの有効活用、そして後から読み返しやすい可読性の高いコードを書くことの重要性を学びました。
